**Goal:** 
To create an interactive insurance policy analysis dashboard on Power BI using a dataset containing records from 2015 to 2025 and extract meaningful insights.


**Description:**
The project is focused on analysing the dataset of TrueSecure Insurance. The dataset includes Customer Info, Agent Info, Policy Type, Fact Table and other relevant information tables. The project involved loading the data, cleaning and pre-processing it, then creating an interactive dashboard according to the client requirement. 


**Skills:**
Data cleaning, Data analysis, Data visualization, DAX


**Results:**
Analytics indicate that the ULIP growth plan has the highest premium paid share at 32.74%. The annual ROI has declined from 0.03% in 2015 to 0.01% in 2024. Additionally, the total premium paid decreased significantly from 1.92 billion in 2016 to 0.31 billion. The states of Delhi, Uttar Pradesh, and Sikkim account for the largest premium payable amounts.
